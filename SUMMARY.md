## Как использовать

### Быстрый старт

1. **Запустите бота:**
   ```bash
   python bot.py
   ```

2. **Установите фильтр в Telegram:**
   ```
   /set_filter trade_frequency_min 10 5
   ```
   (минимум 10 сделок за 5 минут)

3. **Запустите автосканирование:**
   ```
   /auto_scan_start
   ```

### Примеры использования

**Найти очень активные рынки:**
```
/set_filter trade_frequency_min 20 5
/set_filter spread_min 2
/auto_scan_start
```

**Мониторинг конкретного рынка:**
```
/set_market your-market-slug
/set_filter trade_frequency_min 5 3
```

**Комбинация фильтров:**
```
/set_filter spread_min 1.5
/set_filter volume_min 100000
/set_filter trade_frequency_min 10 5
```

## Параметры фильтра

**Формат:**
```
/set_filter trade_frequency_min <количество> <период_минут>
```

**Ограничения:**
- Количество сделок: от 1 и выше
- Период: от 1 до 10 минут

**Примеры:**
- `/set_filter trade_frequency_min 5 1` - очень активные (5 сделок/мин)
- `/set_filter trade_frequency_min 10 5` - активные (10 сделок/5мин)
- `/set_filter trade_frequency_min 15 10` - умеренные (15 сделок/10мин)
