# Быстрый старт: Фильтр частоты сделок

## Что нового?

Теперь бот может фильтровать рынки по частоте совершения сделок! Это позволяет находить только активные рынки с реальной торговой активностью.

## Быстрый старт

### 1. Запустите бота

```bash
python bot.py
```

### 2. Установите фильтр частоты сделок

В Telegram отправьте команду:

```
/set_filter trade_frequency_min 10 5
```

Это означает: показывать только рынки с **минимум 10 сделками за последние 5 минут**

### 3. Запустите автосканирование

```
/auto_scan_start
```

Бот начнет сканировать все рынки и отправлять уведомления только о тех, где есть активная торговля.

## Примеры использования

### Найти очень активные рынки

```
/set_filter trade_frequency_min 20 5
/set_filter spread_min 2
/auto_scan_start
```

### Найти рынки с умеренной активностью

```
/set_filter trade_frequency_min 5 10
/set_filter volume_min 50000
/auto_scan_start
```

### Мониторинг конкретного рынка

```
/set_market your-market-slug
/set_filter trade_frequency_min 3 3
```

Бот будет уведомлять только когда на рынке появляется активность (минимум 3 сделки за 3 минуты).

## Параметры фильтра

**Формат:**
```
/set_filter trade_frequency_min <количество> <период_минут>
```

**Ограничения:**
- Количество сделок: от 1 и выше
- Период: от 1 до 10 минут

**Примеры:**

| Команда | Описание |
|---------|----------|
| `/set_filter trade_frequency_min 5 1` | Минимум 5 сделок за 1 минуту (очень активные) |
| `/set_filter trade_frequency_min 10 5` | Минимум 10 сделок за 5 минут (активные) |
| `/set_filter trade_frequency_min 15 10` | Минимум 15 сделок за 10 минут (умеренные) |

## Комбинирование с другими фильтрами

Вы можете комбинировать фильтр частоты с другими фильтрами:

```
/set_filter spread_min 1.5
/set_filter volume_min 100000
/set_filter yes_price_between 0.3 0.7
/set_filter trade_frequency_min 10 5
```

Бот будет показывать только рынки, которые соответствуют **всем** установленным фильтрам.

## Просмотр и управление фильтрами

### Посмотреть активные фильтры

```
/filters
```

### Удалить фильтр

```
/remove_filter <id>
```

Или нажмите кнопку "Удалить фильтр" в сообщении `/filters`

## Как это работает?

1. Бот запрашивает последние сделки для каждого рынка через Polymarket API
2. Подсчитывает количество сделок за указанный период
3. Сравнивает с вашим минимальным значением
4. Отправляет уведомление только если условие выполнено
5. Результаты кэшируются на 60 секунд для оптимизации

## Советы

💡 **Для поиска новых возможностей:** Используйте низкий порог (5-10 сделок за 5 минут)

💡 **Для стабильных рынков:** Используйте высокий порог (15-20 сделок за 5 минут)

💡 **Для быстрых сделок:** Используйте короткий период (1-3 минуты)

💡 **Для долгосрочного анализа:** Используйте длинный период (8-10 минут)
