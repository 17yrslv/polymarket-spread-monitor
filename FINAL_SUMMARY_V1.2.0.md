## 🚀 КАК ИСПОЛЬЗОВАТЬ

### 1. Запустите бота
```bash
python bot.py
```

### 2. Установите строгие базовые фильтры
```
/set_filter volume_min 500000
/set_filter spread_min 1.5
```
**Эффект:** Отсеет 95% рынков до проверки частоты сделок

### 3. Добавьте фильтр частоты
```
/set_filter trade_frequency_min 5 10
```
**Эффект:** Найдет только активные рынки (минимум 5 сделок за 10 минут)

### 4. Запустите автосканирование
```
/auto_scan_start
```
**По умолчанию:** Сканирует 30,000 рынков

### 5. Проверьте статус
```
/auto_scan_status
/filters
```

### 6. Мониторьте логи
```
tail -f bot.log
```

## 🛡️ ЗАЩИТА ОТ RATE LIMITS

### 1. Увеличенный кэш (5 минут)
- Повторные сканирования используют кэш
- Экономия до 80% запросов

### 2. Обработка ошибки 429
- Автоматическая пауза при rate limit
- Прогрессивное увеличение времени ожидания (10, 20, 30 сек)
- Автоматическое восстановление

### 3. Безопасные задержки
- 0.3 сек между батчами
- 2 сек между страницами
- Не перегружает API

### 4. Оптимальный порядок фильтров
- Быстрые фильтры (spread, volume) идут первыми
- Медленный фильтр (trade_frequency) идет последним
- Минимум API запросов
