## Как использовать

### 1. Запустите бота
```bash
python bot.py
```

### 2. Установите фильтр частоты сделок
```
/set_filter trade_frequency_min 5 10
```
(минимум 5 сделок за 10 минут)

### 3. Запустите автосканирование
```
/auto_scan_start
```

### 4. Проверьте активные фильтры
```
/filters
```

### 5. Ожидайте уведомления
Бот будет присылать уведомления о рынках с активной торговлей!
