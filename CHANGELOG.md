### 2. Новая команда /search

**Использование:**
```
/search Trump
/search Ukraine ceasefire
/search Bitcoin
```

**Что делает:**
- Ищет рынки по ключевым словам
- Показывает до 5 результатов
- Выводит slug для каждого рынка
- Показывает объём торгов

**Пример ответа:**
```
🔍 Найдено рынков: 3

1. Trump out as President before GTA VI?
   Slug: trump-out-as-president-before-gta-vi-846
   Объём: $587,299

2. Trump to visit Ukraine before GTA VI?
   Slug: trump-to-visit-ukraine-before-gta-vi-123
   Объём: $234,567

Для добавления рынка используйте:
/set_market <slug>
```

### 3. Улучшенная команда /set_market

**Что изменилось:**
- Более полезные сообщения об ошибках
- Подсказки при неудаче поиска
- Автоматическое предложение использовать /search

**Пример нового сообщения об ошибке:**
```
❌ Рынок с slug 'wrong-slug' не найден

Возможные причины:

• Рынок закрыт или удалён
• Рынок ещё не загружен в кэш

💡 Попробуйте найти рынок:

```
