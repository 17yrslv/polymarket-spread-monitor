#### ✅ Технические компоненты:

**1. PolymarketAPI класс:**
- Получение списка всех рынков
- Поиск рынка по slug
- Получение актуальных данных рынка
- Расчет bid-ask спреда: `((bestAsk - bestBid) / bestBid) * 100`
- Retry логика: 3 попытки с экспоненциальной задержкой (2^n секунд)

**2. Database класс:**
- Таблица `users` - пользователи бота
- Таблица `markets` - отслеживаемые рынки (до 5 на пользователя)
- Таблица `filters` - фильтры пользователя
- Таблица `monitoring_state` - состояние мониторинга
- Все операции асинхронные через aiosqlite

**3. MonitoringService класс:**
- Фоновый asyncio task для каждого пользователя
- Цикл мониторинга каждые 10 секунд
- Проверка всех рынков пользователя
- Применение фильтров к данным
- Отправка уведомлений в Telegram

**4. Command Handlers:**
- Middleware для проверки User ID
- Валидация входных данных
- Обработка ошибок
- Автоматический запуск/остановка мониторинга

### 🐛 Troubleshooting:

#### Бот не запускается
```bash
# Проверьте установку зависимостей
pip install -r requirements.txt

# Проверьте версию Python
python --version  # Должно быть 3.8+
```

#### Нет уведомлений
```
1. Проверьте статус: /status
2. Проверьте фильтры: /filters
3. Проверьте список рынков: /markets
4. Посмотрите логи в bot.log
```

#### Ошибки подключения
```
1. Проверьте интернет соединение
2. Проверьте прокси настройки в bot.py
3. Убедитесь, что токен бота корректен
```
